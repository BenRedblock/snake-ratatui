## **Building the Project**
### **Prerequisites**
- [rust](https://www.rust-lang.org/)
### **Build Steps**
Clone the repository:
```bash
git clone https://github.com/BenRedblock/snake-ratatui
cd snake-ratatui
```
### Linux
Install Dependencies:
```bash
cargo install
```
Build the project:
```bash
cargo build --release
```
Run the game:
```bash
cargo run --release
```
or
```bash
./target/release/snake-ratatui
```
